# Semantic Search AI Chatbot

## 📌 Overview

This project is a simple AI-based semantic search chatbot. It retrieves the most relevant answer from a dataset based on user queries.

## 🧠 How it Works

* Converts text into vectors using TF-IDF
* Compares similarity using cosine similarity
* Returns the best match with a confidence score

## ⚙️ How to Run

1. Clone the repo
2. Install dependencies:
   pip install scikit-learn
3. Run:
   python app.py

## 💡 Example

Input: What is AI?
Output: Artificial Intelligence is the simulation of human intelligence

## 🚀 Features

* Semantic Search
* Confidence Score
* Simple RAG-based retrieval
